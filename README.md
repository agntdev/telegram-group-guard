# Telegram Group Guard

Telegram bot that greets and verifies newcomers, detects spam, enforces configurable warn/mute/kick/ban actions, and provides admin commands, logs and basic analytics.

Spec: [`docs/blueprint.md`](docs/blueprint.md).

Built on [agnt-gm.ai](https://agnt-gm.ai). The whole bot is built and refined here as pull requests across successive build passes.
